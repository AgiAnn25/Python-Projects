# 🧠 Python Portfolio

This repository contains a collection of projects built using Python. The goal of this repository is to practice real-world datasets, improve data analysis skills, and build strong foundations in machine learning.

---

# 📌 Projects Included

Each project focuses on solving a real-world problem using data analysis, visualization, and machine learning models.

## 🔹 Heart Disease Prediction
A classification project that predicts the likelihood of heart disease based on medical attributes such as age, cholesterol, blood pressure, and chest pain type.

**Key Work:**
- Data preprocessing and encoding
- Exploratory Data Analysis (EDA)
- Feature selection
- Classification model training and evaluation

---

## 🔹 Iris Flower Classification
A classic ML classification problem that identifies flower species based on petal and sepal measurements.

**Key Work:**
- Data visualization and pattern analysis
- Feature scaling
- Model comparison and evaluation

---

## 🔹 Titanic Survival Prediction
A predictive model that determines passenger survival based on features like age, gender, class, fare, and family size.

**Key Work:**
- Handling missing values
- Encoding categorical variables
- Correlation analysis
- Classification modeling

---

## 🔹 Ola_Fare_Prediction
A time-series and demand analysis project focused on understanding ride booking patterns based on time, season, and external factors.

**Key Work:**
- Time-based feature engineering
- Data visualization
- Trend analysis
- Predictive modeling
