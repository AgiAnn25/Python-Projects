# Heart Disease Prediction ❤️

## Overview
The Heart Disease Prediction project aimed to predict whether a person has heart disease using Machine Learning techniques. The dataset included medical features such as age, sex, chest pain type, cholesterol, blood pressure, ECG results, and more.

## Data Preprocessing
Since there were no missing values in the dataset, no filling or replacement of missing data was required. All categorical features were converted into numerical form using encoding techniques to make the dataset suitable for machine learning model training.

## Exploratory Data Analysis
Data visualization techniques such as count plots and scatter plots were used to analyze patterns and relationships in the dataset. The analysis showed that heart disease was more common among females and individuals with lower maximum heart rates.
